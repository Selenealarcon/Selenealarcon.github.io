Proyectos

PlanAhead

Desarrollo de una aplicación Laravel que facilita la gestión de tareas y la colaboración entre usuarios en diversos espacios. Esta plataforma permite crear y personalizar espacios virtuales donde los usuarios pueden organizar y asignar tareas, optimizando así la coordinación en contextos como el hogar, la escuela o el trabajo.

  • Laravel
  • HTML
  • CSS
  • jQuery
  • UML
  • MySQL

MyRestaurant

Recreación de la web de un restaurante, centrada en lograr un diseño responsivo que se ajuste de forma automática a distintos dispositivos, proporcionando una experiencia de usuario óptima en cualquier pantalla.

  • HTML
  • CSS
  • Figma

Bingo

Desarrollo de una aplicación de Bingo que simula el tradicional juego en un entorno digital, ofreciendo a los usuarios la capacidad de generar cartones de forma aleatoria y jugar en tiempo real. La aplicación se basa en un algoritmo de generación aleatoria que asegura que cada cartón sea único, utilizando estructuras de datos eficientes para almacenar y gestionar los números.

  • Java

Spotify

Recreación de una réplica de la página de Spotify utilizando HTML y CSS, incorporando media queries para lograr un diseño responsivo que se adapte a distintos dispositivos. La estructura incluye una página de inicio de sesión que redirige a la página principal, donde se presentan diversos podcasts y álbumes. También se ha diseñado una página dedicada a un álbum específico, mostrando una lista de canciones y detalles del mismo.

  • HTML
  • CSS

Conecta 4

Desarrollo de una aplicación de Conecta 4 que recrea el clásico juego de mesa en un entorno digital con una interfaz de consola. La aplicación permite que dos jugadores coloquen fichas en un tablero virtual, verificando automáticamente las condiciones de victoria (horizontal, vertical y diagonal) y los empates. Un algoritmo aleatorio determina qué jugador inicia cada partida, garantizando una experiencia dinámica, mientras que estructuras de datos eficientes gestionan las posiciones de las fichas y el recuento de victorias.

  • Java
